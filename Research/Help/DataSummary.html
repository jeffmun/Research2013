Help for Data Summaries

Data managed by the Statistics Core will be made available through this portion of the website. This page as well as the "Documents - Measures by Study" report in the Data Documentation section of the Report Library will give you access to all the relevant documents that describe the measure.

The columns of this report are as follows:

Data Entry Form Link to the data entry form if available.
Data Form / Var Map PDF copy of the data recording form for the measure, annotated with the corresponding database fieldnames for each item (a.k.a. variable map).
Field Info / Data Dict Data dictionary for the measure. This is a list of all the fields in the database table, labels, missing value codes, valid values, and value labels (a.k.a. data dictionary).
Admin Protocol Link to the administration protocol for the measure if available.
Coding Manual Link to the coding manual if available.
Stored Proc Link to a report displaying any SQL stored procedures used to score the measure.
SQL Functions Link to a report displaying any SQL functions used to score the measure.
SPSS syntax Link to the SPSS syntax file used to score the measure.
Desc Stats (last run) Displays the descriptive statistics for the data: Means, etc., histograms, scatterplots, boxplots for relevant variables for the full sample and by sex and group. The value in parentheses displays how long ago the descriptive stats were generated.
SPSS Full Link to download SPSS .sav file of every variable entered for the measure.
SPSS Brief Link to download SPSS .sav file of only those variables typically needed for data analysis.
Tab-delim Full Link to download tab-delimited ascii file of every variable entered for the measure. Copy and paste or save to file as needed.
Tab-delim Brief Link to download tab-delimited ascii file of only those variables typically needed for data analysis. Copy and paste or save to file as needed.
Staff Data Staff member assigned to complete the processing of the measure.
